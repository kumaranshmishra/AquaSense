About Aqua Sense

How it works — from multiple devices to real-time insights

📡

Multi-Device Network

Connect multiple Arduino sensors across different locations. Each device has its own data stream and can be monitored independently.

🐍

Centralized Backend

Flask server manages multiple serial connections, aggregates data from all devices, and serves unified API endpoints.

📊

Device Management

Easily add, remove, and switch between devices. Each device maintains its own history and can be configured independently.

🌿

Scalable Solution

From one sensor to hundreds, Aqua Sense scales with your monitoring needs. Perfect for river networks and industrial applications.
